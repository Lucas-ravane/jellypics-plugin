JellyPics Upload

Configurez le dossier de destination pour la synchronisation automatique depuis l'application JellyPics. Ce dossier doit exister sur le serveur et être inclus dans une médiathèque Jellyfin.

Les photos synchronisées depuis JellyPics arriveront dans ce dossier. Laissez vide si vous ne souhaitez utiliser que l'import sélectif.
